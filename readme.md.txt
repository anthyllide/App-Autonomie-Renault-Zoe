Propriétaire d'une Renault Zoé, je souhaitais pouvoir visionner facilement sur mon smartphone le simulateur 
d'autonomie Renault. 
Mais, ce dernier a été conçu pour fonctionner plutôt sur PC. Etant développeuse web de métier, j'ai créé cette application 
en utilisant les données de Renault, trouvées sous forme de "json" dans le code source du simulateur d'origine. 

Côté langage, l'application a été développée en utilisant le framework AngularJS et Cordova pour son déploiement.

Remarque : De 2012 (année de première sortie) à 2016, les voitures Renault Zoé étaient équipées d'une batterie de 22kWh.
Les dernières Zoé disposent d'une batterie avec une capacité double de 41kWh. Afin de satisfaire tous les propriétaires, 
l'application prend en compte les deux types de batteries.
